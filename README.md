## ✨ Features

| Feature | Description |
|---------|-------------|
| ⚡ **Real-time Counting** | Word, character, sentence & paragraph counts update instantly as you type |
| 📖 **Reading Time** | Estimates reading time based on 238 WPM (research-backed average) |
| 🎤 **Speaking Time** | Calculates speech duration at 130 WPM for presentations |
| 🔑 **Keyword Density** | SEO analysis showing most frequent words (excludes stop words) |
| 📋 **Copy & Download** | One-click copy to clipboard or download as .txt file |
| 🔒 **100% Private** | All processing happens in your browser — text never leaves your device |
| 📱 **Fully Responsive** | Works perfectly on mobile, tablet, and desktop |
| 🚫 **Zero Ads** | Clean interface with zero distractions |

## 📊 Stats Dashboard

The app displays 8 real-time metrics:

1. **Words** — Total word count
2. **Characters** — With spaces
3. **Characters (no spaces)** — Clean character count
4. **Sentences** — Automatic sentence detection
5. **Paragraphs** — Block-level analysis
6. **Reading Time** — Minutes to read silently
7. **Speaking Time** — Minutes to present aloud
8. **Avg Word Length** — Average characters per word
